# Research And Scoping

Use for evidence planning, claim maps, source quality, scope boundaries, and deciding whether enough public support exists.

## Process

1. Turn the thesis into claims.
2. Mark each claim as factual, interpretive, practical, speculative, or normative.
3. Attach public sources or source gaps to each claim.
4. Separate examples from evidence.
5. Define what the article will not cover.
6. Identify freshness risks for fast-moving AI, policy, product, or model claims.

## Source Rules

- Public sources only.
- Prefer primary sources for technical claims.
- Use recent sources for unstable topics.
- Name weak evidence explicitly rather than hiding it.
- Route source quality disputes to `source-challenge`.

## Output

Return a claim map, source map, scope boundary, open research gaps, and risk notes.
