# Visualize Prompt

Decide whether this proposal needs visual support.

Prefer visuals only when they clarify:

- sequence
- comparison
- system structure
- taxonomy
- evidence flow
- a repeatable workflow

Return one of:

- `visual_plan`: a concise description of the useful visual
- `example_plan`: examples are enough
- `no_visual_reason`: why visuals would be decorative or distracting

Keep visuals public-safe and free of client, project, proprietary, or personal details.
