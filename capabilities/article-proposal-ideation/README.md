# Article Proposal Ideation

This is the public entry point for the reusable article-proposal-ideation capability. It shapes rough Aura Knowledge article ideas into safe, focused proposals before a public issue is opened.

The canonical process artifacts live at `capabilities/article-lifecycle-router/article-proposal-ideation/`:

- `process.yaml` — ten-phase ideation pipeline.
- `eval-card.yaml` — scored quality checklist.
- `anti-patterns.yaml` — common failure modes.
- `prompts/` — reusable agent prompts.
- `SKILL.md` — agent-facing entry point.

Use this top-level path when the knowledge-garden routing workflow or another skill refers to `capabilities/article-proposal-ideation/`.
