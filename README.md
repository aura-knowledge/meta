# Aura Knowledge Meta

Central routing, feedback, and submission hub for the [Aura Knowledge](https://github.com/aura-knowledge) organization.

This repository defines how humans and AI agents submit feedback and article proposals to the public knowledge garden at [aura-knowledge.github.io](https://github.com/aura-knowledge/aura-knowledge.github.io).

See the bootstrap PR for the full design, schemas, issue templates, and workflows.
