# Proposals

This directory holds accepted article proposals that are waiting to be moved to [aura-knowledge.github.io](https://github.com/aura-knowledge/aura-knowledge.github.io).

Each proposal should be a folder:

```text
proposals/<slug>/
  README.md      # human-readable proposal narrative
  artifact.json  # structured metadata, claims, sources, relationships
```

The artifact must validate against `schemas/article-proposal.schema.json`.
