# The Modern AI Stack: Demystifying the Architecture of Artificial Intelligence

**Issue:** [#54](https://github.com/aura-knowledge/meta/issues/54)

**Maturity:** sprout

This proposal introduces a nine-layer mental model for understanding the modern AI ecosystem, from compute hardware and base models through alignment, inference, retrieval, orchestration, tool protocols, guardrails, and end-user applications. The full draft is in [`article.md`](./article.md).

## Key claims

- Modern AI systems are best understood as a modular stack rather than a single monolithic model.
- Base-model scaling remains important, but much of the visible product engineering and investment has moved up the stack toward orchestration and tool protocols.
- Retrieval-augmented generation is a common pattern for grounding models in proprietary or real-time data.
- Model Context Protocol (MCP) is emerging as a common adapter for agent-tool integration.
- End-user applications such as Claude Desktop bundle multiple lower layers into a single GUI.
